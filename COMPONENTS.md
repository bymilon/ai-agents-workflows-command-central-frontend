# Components Library

## Philosophy
- Fully headless primitives wrapped with customized styling strings (`tailwindcss` + `cva`).
- Radix-ui inspired.

## Standard Set
- `Button`: Multiple variants (`primary`, `ghost`, `danger`) all maintaining precise radii (`14px`) and glow states.
- `Panel`: Elevated nested containers relying on luminance shifts (`background: #0D0F17`) rather than drop shadows.
